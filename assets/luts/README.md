# LUT Files (.cube)

Place your `.cube` LUT files in this directory. They will be automatically bundled into the iOS and Android native builds and appear in the story editor's filter picker.

## Naming Convention

The filter name displayed in the UI is derived from the filename (minus the `.cube` extension).
For example: `Sedona.cube` → "Sedona", `Blade_Runner_2049.cube` → "Blade Runner 2049"
